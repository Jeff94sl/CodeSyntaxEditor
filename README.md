# CodeSyntaxEditor
Es un editor de texto plano orientado a la programación de codigo abrierto desarrollado en csharp 
y [GtkSharp](http://www.mono-project.com/docs/gui/gtksharp/).

# Compilación
Para la corecta compilacion del programa recomendamos usar [Monodevelop](https://www.monodevelop.com/),
siga los pasos de instalacion que en la pagina oficial se muestran.

# Ejecucion
Cree una mueva entrada de menu y agrage la siguente orden:
mono CodeSyntaxEditor.exe

ó
desde la terminal usando el mismo comando
